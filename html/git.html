Git

  • git init - Инициализация репозитория
  • git clone "ссылка с GitHab" - Клонировать удалённый репозиторий с GitHab себе на компьютер
  • git status - Просмотр изменений
  • git add "имя файла" - Добавить файл в отслеживаемые
  • git commit -m "наименование" - сделать коммит
  • git push - Внести изменения на удалённый репозиторий
  • git log - Проверить изменения
  • git pull - Внести изменения из удалённого репозитория в локальный
